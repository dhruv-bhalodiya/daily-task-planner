# Daily Task Planner Website


This project is a simple, offline daily task management website built using HTML, CSS, and JavaScript to help users organize and complete their daily activities efficiently.

---

## Tech Stack

- **HTML** – Used to create the structure of the website including input fields, buttons, and task lists.
- **CSS** – Used for styling the UI, layouts, colors, responsiveness, and simple animations.
- **JavaScript** – Used to handle logic such as adding tasks, deleting tasks, saving data, and reminders.
- **LocalStorage** – Used to store tasks locally so data remains after page refresh.

---

## Features

### 1. Add Daily Task
Allows the user to add a task with a title, time, and priority.
The task details are captured through input fields and stored as an object.

---

### 2. Task List Display
Displays all added tasks in a clean list format.
Each task shows its name, scheduled time, priority level, and status.

---

### 3. Task Priority System
Provides priority selection (Low, Medium, High) for each task.
Different colors are used to visually distinguish task importance.

---

### 4. Mark Task as Completed
Allows users to mark tasks as completed using a checkbox or button.
Completed tasks appear with a strike-through and move to the bottom.

---

### 5. Delete Task
Provides an option to delete individual tasks easily.
The task is removed from both the UI and localStorage.

---

### 6. LocalStorage Data Saving
Automatically saves all tasks in the browser’s localStorage.
Tasks remain available even after refreshing or closing the browser.

---

### 7. Daily Task Reset
Checks the current date and resets tasks for a new day.
Prevents old tasks from mixing with the new day’s schedule.

---


### 8. Responsive Design
Ensures the website works smoothly on mobile, tablet, and desktop screens.
Uses flexible layouts and media queries for better usability.

---

### 9. Clean & Simple UI
Provides a distraction-free interface focused on productivity.
Uses minimal colors, readable fonts, and smooth animations.
